Problemas Pendientes

Estos problemas aún necesitan atención para maximizar el potencial SEO del sitio.

1. Meta Descriptions Ausentes en Páginas Internas CRÍTICO

Problema: Solo la homepage tiene meta description. Las demás páginas carecen de ella:

  • ❌ /carta - Sin meta description
  • ❌ /reservas - Sin meta description
  • ❌ /sobre-nosotros - Sin meta description
  • ❌ /menus-especiales - Sin meta description
  • ❌ /eventos - Sin meta description
  • ❌ /contacto - Sin meta description

Impacto: Pérdida de CTR en resultados de búsqueda para estas páginas importantes.

2. Canonical Tags Ausentes CRÍTICO

Problema: Ninguna página tiene etiqueta canonical definida.

Impacto: Riesgo de contenido duplicado y dilución de autoridad de página.

3. Open Graph y Twitter Cards Ausentes CRÍTICO

Problema: No hay meta tags para compartir en redes sociales en ninguna página.

Impacto: Cuando alguien comparte el sitio en Facebook, WhatsApp, Twitter, etc., no aparece imagen ni descripción optimizada.

4. Schema Markup Solo en Homepage IMPORTANTE

Problema: Las páginas internas no tienen structured data específico:

  • ❌ /carta - Falta schema Menu con precios
  • ❌ /eventos - Falta schema Event
  • ❌ /menus-especiales - Falta schema AggregateOffer

5. Alt Text Genéricos IMPORTANTE

Problema: Aunque se añadieron alt text, son demasiado genéricos ("Sal Negra Restaurant").

Recomendación: Usar descripciones específicas por imagen (ver ejemplos abajo).

6. Horarios de Apertura Sospechosos VERIFICAR

Detectado: Los horarios en el schema indican "09:00 - 17:00" todos los días.

Problema: Esto es inusual para un restaurante de mariscos (típicamente abierto para cenas).

Acción: Verificar y actualizar con horarios reales del restaurante.

7. Sin Hreflang Tags MEDIO

Problema: No hay indicación del idioma del sitio para buscadores.

Recomendación: Añadir hreflang="es" como mínimo.
